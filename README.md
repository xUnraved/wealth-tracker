# Wealth Tracker

Projekt für das Modul Webtechnologien – HTW Berlin, SoSe 2026

## Beschreibung
Ein persönlicher Vermögens-Tracker, mit dem man Aktien, ETFs, Krypto, 
Bargeld und andere Werte verwalten und den Überblick über sein Vermögen behalten kann.

## Tech-Stack
- Backend: Spring Boot (Java)
- Frontend: Vue.js (folgt)
- Datenbank: PostgreSQL (folgt)
- Deployment: Render.com (folgt)

## API
| Method | Endpoint | Beschreibung |
|--------|----------|--------------|
| GET | /assets | Gibt alle Assets zurück |

## Starten
./gradlew bootRun
